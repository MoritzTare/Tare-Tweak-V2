Tare Tweak v2

Windows Optimization & Debloat Utility

Über Tare Tweak v2

Tare Tweak v2 ist ein interaktives Windows-Batch-Tool zur Systemoptimierung, Leistungsverbesserung und Reduzierung unnötiger Hintergrunddienste. Alle Funktionen sind über ein menügesteuertes Konsoleninterface erreichbar.

Beim Start wird automatisch ein Systemwiederherstellungspunkt erstellt, um Änderungen jederzeit rückgängig machen zu können.

Funktionen

  • ✔ Windows-Settings-Optimierung
  • ✔ Umfassende Telemetrie- & Datenschutz-Tweaks
  • ✔ Debloat-Funktionen (Windows-Apps entfernen / wiederherstellen)
  • ✔ CPU- & Energieprofil-Optimierungen
  • ✔ I/O- & Eingabeverzögerungs-Tweaks
  • ✔ Temp- & Cache-Bereinigung
  • ✔ Anpassung der Mouse / Keyboard Data Queue Size

Sicherheit

Das Skript verwendet ausschließlich native Windows-Befehle (Batch, Registry, PowerShell). Es wird keine externe Software installiert und es findet keine Datenübertragung statt.
