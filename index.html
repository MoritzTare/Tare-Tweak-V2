Tare Tweak v2

Windows Optimization & Debloat Utility

About Tare Tweak v2

Tare Tweak v2 is an interactive Windows batch tool for system optimization, performance enhancement, and reducing unnecessary background services. All features are accessible via a menu-driven console interface.

A system restore point is automatically created on start so all changes can be reverted at any time.

Features

  • ✔ Windows Settings Optimization
  • ✔ Comprehensive Telemetry & Privacy Tweaks
  • ✔ Debloat Functions (Remove/Restore Windows Apps)
  • ✔ CPU & Power Profile Optimizations
  • ✔ I/O & Input Delay Tweaks
  • ✔ Temp & Cache Cleaning
  • ✔ Mouse / Keyboard Data Queue Size Adjustment

Security

This script only uses native Windows commands (Batch, Registry, PowerShell). No external software is installed and no data is transmitted.
