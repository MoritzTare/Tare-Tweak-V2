Tare Tweak v2

Windows Optimization & Debloat Utility

About Tare Tweak v2

Tare Tweak v2 is an interactive Windows batch tool for system optimization, performance enhancement, and reduction of unnecessary background services. All features are accessible through a menu-driven console interface.

Upon launch, a system restore point is automatically created so all changes can be reversed at any time.

Features

  • ✔ Windows settings optimization
  • ✔ Comprehensive telemetry & privacy tweaks
  • ✔ Debloat functions (remove/reinstall Windows apps)
  • ✔ CPU & power profile optimizations
  • ✔ I/O & input latency tweaks
  • ✔ Temp & cache cleaning
  • ✔ Mouse / keyboard data queue size adjustments

Security

The script only uses native Windows commands (Batch, Registry, PowerShell). No external software is installed, and no data is transmitted.
